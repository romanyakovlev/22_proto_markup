# Что использовал
- Bootstrap
- Jinja2(staticjinja)
- Browsersync
- W3C validator
- YSlow
- GitHub Pages

# Запуск
Клонируем репозиторий, заходим в него, устанавливаем зависимости:
``
pip install -r requirements.txt
``
Запускаем скрипт:
``
python3 run.py
``
